# Sudarsan G — Portfolio

A responsive, single-page portfolio highlighting AI/ML projects, IoT builds, education, internships, and certifications for Sudarsan G. Built with vanilla HTML, CSS, and JavaScript.

## Structure
- `index.html` — main page structure
- `styles.css` — theme, layout, animations
- `script.js` — navigation toggle and header behavior

## Run locally
1. Open `index.html` in your browser, or
2. Serve the folder via a simple HTTP server (e.g., `python -m http.server 8000`) and visit `http://localhost:8000`.

## Customization
- Update contact links, copy, or projects directly in `index.html`.
- Adjust colors and spacing in `styles.css` via the CSS variables at the top.
- Extend interactivity in `script.js` as needed.
